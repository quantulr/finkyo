# finkyo

## How to use.

### Serve a directory:
    finkyo ./
### Use custom port:
    finkyo ./ -p 8080

## Usage
```
Usage: finkyo.exe [OPTIONS] [PATH]

Arguments:
  [PATH]  [default: ./]

Options:
  -p, --port <PORT>  [default: 3000]
  -c, --compress     enable directory compress
  -h, --help         Print help
  -V, --version      Print version
```
